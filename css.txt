These are brief descriptions of things I learned going through this MDN
module:
https://developer.mozilla.org/en-US/docs/Learn/CSS/First_steps

=========

1. What is CSS?
[https://developer.mozilla.org/en-US/docs/Learn/CSS/First_steps/What_is_CSS]

Defines what kind of language CSS is. It also instructs where to find
reference to various properties, modules and the spec. Also includes a video
about the default browser styles.

========

2. Getting Started with CSS
[https://developer.mozilla.org/en-US/docs/Learn/CSS/First_steps/Getting_started]

Provides a very detailed and high level overview of applying CSS to the HTML.
Covers selectors, styling based on where tags are in the HTML, the state of a
certain tag etc.

=======
3. How CSS is structured
[https://developer.mozilla.org/en-US/docs/Learn/CSS/First_steps/How_CSS_is_structured]

Goes into more details about selectors, property-values, @rules, shorthands
etc.

=======

4. How CSS works
[https://developer.mozilla.org/en-US/docs/Learn/CSS/First_steps/How_CSS_works]

Provides a very nice overview of how CSS is parsed and then rendered by the
browser.

These are brief descriptions of things I learned going through this MDN
module:
https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks

=========

1. Cascade and Inheritance (& Specificity)
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Cascade_and_inheritance]

A brief description on inheritance, cascade and specificity. Explains really
well how the browser calculates which style to apply to an element. Provides a
really good description of the 'inherit' value and how to use it well.

=========

2. CSS Selectors
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Selectors]

Provides brief intro to various types of selectors available in CSS. Contains
a reference on most of these selectors and links to articles that talk about
them more throughly.

--

2.1 Type, Class and ID Selectors
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Selectors/Type_Class_and_ID_Selectors]

Covers type/element selectors, the universal selector, class and ID selectors.

--

2.2 Attribute Selectors
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Selectors/Attribute_selectors]

Covers selecting tags using attr selectors. They look like: p[class='a'].
Quite a useful article that covers matching them using regex and other
natively available ways.

-- 

2.3 Pseudo-classes and pseudo-elements
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Selectors/Pseudo-classes_and_pseudo-elements]

Introduces pseudo-classes and pseudo-elements. Explains really well what they
are and when/how to use them. Provides a reference of them in the end of the
article.

--

2.4 Combinators
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Selectors/Combinators]

Describes some very useful combinators to be familiar with: Descendant, Child,
Adjacent sibling and General sibling.


========

3. The box model
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/The_box_model]

Describes the box model throughly. The box model is how the boxes (elements)
of in a page are defined and behave. There is the default and the alternate
box model.

7. Values and Units
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Values_and_units]

A really important article that introduces how em, rem and % work. Also it
defines what 'datatypes' or 'value types' are and describes the most commonly
implemented datatypes. 

======

8. Sizing Items in CSS
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Sizing_items_in_CSS]

Article discusses how to size things using CSS. From setting specific sizes
and their pitfalls to using min- max-, viewport units etc. It also talks about
the intrinsic and extrinsic sizes of elements.

=====

9. Images, media, and form elements
[https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Images_media_form_elements]

Talks about sizing images and form elements. These are called replaced
elements as css cannot alter their internal dimensions.
