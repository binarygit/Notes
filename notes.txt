Git

- delete last commit: git reset --hard HEAD^
- rename branch: git branch -m <new name>
- get number of commits between start commit & 
  end: git rev-list <start-hash>...<end-hash> | wc -l
- delete remote branch: git push origin :<branch-name>
  eg: git push origin :new-landing
- amend last commit: git commit --amend
  when you want to add files that you forgot, 
  just stage them and they will be picked up
